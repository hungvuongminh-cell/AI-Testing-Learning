### 📌 Project 0002: Multi-LLM Logical Reasoning Evaluator
**Goal:** Benchmark the logical consistency and instruction-following of top-tier LLMs (Llama 3.3, Gemma 2, Mixtral) using complex "trap" scenarios.

**Key Features:**
* **Structured Prompting:** Implemented a modular prompt framework (Role, Task, Context, Constraints).
* **Cross-Model Comparison:** One-click model switching via `MODEL_INDEX` to evaluate response quality vs. latency.
* **Anti-Hallucination Guardrails:** Strict instructions to prevent AI from "making up" information.

**Status:** ✅ Completed - Successfully integrated Groq Cloud with LangChain.
